# Model v1 – Diabetes Prediction

- Dataset: Pima Indians Diabetes
- Algoritmo: Random Forest
- Pré-processamento:
  - Imputação de zeros pela mediana
  - Normalização MinMax
- Métrica principal: ROC-AUC
- Threshold de decisão: 0.30

## Limitações
- Dataset pequeno
- Dados antigos
- Não inclui variáveis clínicas modernas

## Próximos passos
